## Setup

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   ```

2. Install the requirements:
   ```bash
   pip install -r requirements.txt
   ```

3. Install Playwright:
   ```bash
   pip install playwright
   ```

4. Run the project:
   ```bash
   python main.py
   ```
